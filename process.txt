————————로그인 상세기능정의————————
주소창에 URL치고 -> 맨 처음 로그인 화면 출력 -> 회원가입 클릭 -> 개인정보를 입력 후 회원가입 -> 서버에서 아이디와 비밀번호를 DB에 저장 -> 학생이 자신의 아이디와 비밀번호로 로그인 시도 -> 로그인 시도 시, DB에서 해당 ID를 검색하고 ID와 비밀번호가 일치하는지 확인 -> 로그인 된 화면 출력

—————————————————댓글 프로세스———————————————
댓글 기능 정의 : 로그인 한 후 수강 한 과목 클릭 -> 서버에 해당 과목 페이지 출력 요청 -> 서버에서 해당 과목 데이터베이스 조회하여 과목에 해당하는 페이지 출력 -> 게시물 클릭(서버에 게시물 페이지 출력 요청) -> 게시물에 해당하는 DB조회하여 해당 페이지 출력 -> 댓글을 작성 한 후 작성하기 클릭 -> 서버에 댓글의 정보 전달 -> 서버는 전달 된 정보를 토대로 DB에 저장하고 댓글을 해당 게시물에 출력

ㄴ> 수정 프로세스(작성한 본인에게만) : 작성한 댓글에 달려있는 수정 버튼 클릭 -> 해당 댓글을 텍스트박스로 교체(기존 댓글은 유지한 상태) -> 수정을 완료하고 수정완료 클릭 -> 수정된 댓글을 DB에 UPDATE

ㄴ> 삭제 프로세스(작성한 본인에게만) : 작성한 댓글에 달려있는 삭제 버튼 클릭 -> 삭제를 클릭하면 확인 팝업 출력 
-> 확인버튼을 클릭하면 DB에서 해당 댓글을 DELETE한 후 댓글 상태 DB를 반영한(댓글 삭제된) 페이지 새로 출력
ㄴ> 아니오를 클릭하면 아무것도 하지 않음


ㄴ> 답글 프로세스 : 댓글에 달려있는 답글 버튼 클릭 -> 댓글작성창을 화면에 출력(기존에 있던 댓글창 작성창과는 다른) -> 답글 작성 -> 서버에 답글의 정보 전달(이 때 해당 댓글의 자식형태로 들어가게 depth를 설정) -> 답글 출력


————————————————강좌 개설—————————————————
강좌 개설 정의 : 로그인을 한 후 강좌개설 버튼 클릭 -> 강좌의 필요 정보를 입력한 후 강좌를 개설

TA강좌 승인 : 로그인을 한 후 TA승인 탭을 클릭 -> 승인 된 강좌와 미승인 된 강좌가 출력되고 미승인 된 강좌란에 있는 승인 버튼 클릭하면 해당 강좌에 대한 TA의 권한을 TA권한으로 설정


————————————————아이디 / 비밀번호 찾기——————————————
아이디 찾기 : 아이디 찾기 버튼 클릭 -> 찾으려는 아이디를 DB에서 찾기 위해 학번과 이름을 입력받음 -> 해당 학번과 이름을 학생목록 DB에서 조회하여 ID를 보여줌



비밀번호 찾기 : 비밀번호 찾기 버튼 클릭 -> 비밀번호 찾기 페이지에서 자신이 찾으려는 아이디를 입력 -> 그 외에 본인의 신상을 증명할 수 있을만 한 정보들을 입력 -> 맞다면 비밀번호를 새로 입력하는 창으로 넘어감 -> 새로운 비밀번호로 해당 ID의 DB를 업데이트

ㄴ> 틀리다면 틀렸다는 경고문을 출력 후 아무것도 하지 않음



—————————————————파일업로드—————————————————

파일 업로드 : 원하는 게시글 찾아들어가서 제출 버튼 클릭 -> 제출페이지가 출력되고 파일 선택하기 버튼 클릭 -> 탐색기를 실행하여 원하는 파일을 선택하게 함 -> 선택 된 파일을 제출 할 파일 목록에 등록 후 제출하기 버튼 클릭하면 파일을 서버에 업로드하고 제출 된 파일에 해당 파일을 출력


———————————————파일 다운로드————————————————

파일 다운로드 : 해당 게시글 찾아들어가서 제출 버튼 클릭 -> 제출 된 파일에서 원하는 파일 선택해서 클릭하면 자동으로 다운로드


———————————————제출한 파일 삭제——————————————

파일 삭제 : 게시글을 찾아 들어가서 제출 버튼 클릭 -> 제출 페이지가 출력 -> 제출 된 파일에 있는 삭제할 파일을 선택한 후 삭제 클릭 -> 확인 메세지를 출력 -> 확인을 누르면 해당 파일을 삭제
							ㄴ> 아니오를 누르면 아무것도 하지 않음


———————————————시험등수 & 등위——————————————
성적을 입력받고 나서 서버에서 DB에 입력된 시험에 해당하는 점수를 sort하여 해당 점수에 해당하는 INDEX를 시험 등수로 등록하게 함(단, sort할 때 같은 점수라면 다 같은 등수로 매겨야함 ex. 5점 , 4점, 4점, 3점이 있을 때, 5점 == 1등 , 4점 = 공동2등 , 3점 = 4등 이런식으로)



————————————————글 작성하기—————————————————

교수님 & TA 글 작성하기 (과제 및 공지사항): 로그인 후 해당 교과목 클릭 -> 해당 교과목에 올린 자료와 공지사항 등등 출력, 없다면 빈 화면을 출력(진짜 아무갓도 없는 빈 화면 말고 목록은 있는데 깨끗한 화면) -> 작성하기 버튼 클릭 -> 입력받을 내용들을 모두 입력받고 옵션을 설정한 후 저장하기 클릭(옵션으로는 말머리, 제출기한 등) -> 위에 입력받은 내용으로 DB에 저장 및 업데이트 후 초기화면(강좌를 클릭했을 때 나오는 화면)에 방금 입력한 내용이 업데이트 된 채 출력

————————————————강좌 신청—————————————————

강좌 신청 : 로그인 후 강좌 신청 클릭 -> 이미 개설되어있는 강좌목록을 화면에 출력 -> 체크박스에 원하는 강좌를 체크한 후 신청 버튼 클릭 -> DB에서 해당 강좌에 학생을 업데이트 하고, 학생에 강좌id를 업데이트

TA강좌신청 : 로그인 후 강좌 추가 클릭 -> 담당 교수님의 강의를 선택하여 교수님께 승인 요청 -> 미승인란에 해당 강의를 출력 -> 교수님이 승인하면 승인란으로 해당 강좌를 이동 후 출력

강좌 취소 : 로그인 후 내 강좌에서 편집 클릭 -> 취소하고 싶은 강좌란에 있는 삭제버튼 클릭 -> 확인을 위한 팝업창에서 확인을 클릭하면 해당 강좌를 학생 DB에서 지우고, 강좌 DB에서는 해당 학생을 삭제 -> DB를 반영하여 출력
ㄴ> 아니오를 클릭하면 아무것도 하지 않음

————————————————채점하기—————————————————
TA & 교수 채점하기 : 로그인 후 채점 할 강좌에 있는 글에 있는 채점하기 버튼 클릭 -> 학생들에게 과제에 해당 점수 입력 후 저장버튼 클릭 -> DB에 반영하여 모든 학생들에게 점수 부여하고 해당 학생에게 점수 출력



————————————————타임라인—————————————————

타임라인 메인화면에 출력(학생용) : 로그인 -> 해당 학생이 듣고 있는 강좌에 있는 게시글을 조회하여 화면에 출력

타임라인 메인화면에 출력(교수용) : 로그인 -> 개설한 강좌에 있는 학생 게시판에 있는 게시글을 조회하여 화면에 출력
